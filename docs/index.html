Core Features

Everything you need to secure your network

A complete security toolkit that runs on your own infrastructure, giving you full visibility and control.

Device Discovery

ARP-based network scanning with nmap detects every device on your LAN. Identifies hostname, vendor, MAC address, and device type automatically.

Port Scanning

Automated port scanning across all discovered devices. Detects open ports, running services, product versions, and flags suspicious configurations like RDP, Telnet, and SMB.

Traffic Analysis

Captures and analyzes network traffic patterns. Shows top destinations, protocol distribution, and GeoIP mapping of all external connections.

Anomaly Detection

Intelligent engine detects suspicious activity: connections to watchlist countries, unusual port usage, traffic spikes, and new unknown devices joining the network.

Real-Time Alerts

WebSocket-powered live alert feed with severity classification (Critical / High / Medium / Low). Acknowledge, resolve, or bulk-dismiss alerts from the dashboard.

Interactive Dashboard

Dark-themed React dashboard with live charts, network topology map, device inventory table, traffic graphs, and full settings management.

Quick Start

Up and running in 2 minutes

One Docker Compose command and your entire stack is live.

1

Clone the repository

git clone https://github.com/SaharHalili95/netsentinel.git cd netsentinel
2

Configure environment

Copy the example env file and set your network range.

cp .env.example .env # Edit SCAN_NETWORK=192.168.x.0/24 to match your LAN
3

Launch with Docker Compose

docker-compose up
4

Open the dashboard

Navigate to http://localhost:3000 — NetSentinel will start scanning your network automatically.
