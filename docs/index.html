Core Features

Everything you need to secure your network

A complete self-hosted security toolkit with full visibility and real-time control.

Device Discovery

ARP-based network scanning with nmap detects every device on your LAN — hostname, vendor, MAC address, and device type.

Port Scanning

Automated scanning of all discovered devices. Detects open ports, services, versions, and flags risky configurations like RDP, Telnet, and SMB.

Traffic Analysis

Captures traffic patterns in real time. Shows top destinations, protocol distribution, and GeoIP mapping of all external connections.

Anomaly Detection

Detects suspicious activity: connections to watchlist countries, unusual ports, traffic spikes, and new unknown devices joining the network.

Real-Time Alerts

WebSocket-powered live alert feed with severity classification. Acknowledge, resolve, or bulk-dismiss alerts directly from the dashboard.

Dark Dashboard

React-based dashboard with live charts, network topology, device inventory table, traffic graphs, and comprehensive settings management.

Quick Start

Up and running in 2 minutes

One Docker Compose command and your entire stack is live.

1

Clone

git clone https://github.com/SaharHalili95/netsentinel.git cd netsentinel
2

Configure

Set your network range in .env

cp .env.example .env # Edit SCAN_NETWORK=192.168.x.0/24
3

Launch

docker-compose up
4

Open dashboard

Navigate to http://localhost:3000 — scanning starts automatically.
