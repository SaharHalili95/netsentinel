Core Features

Everything you need to secure your network

A complete self-hosted security toolkit with full visibility and real-time control.

Device Discovery

ARP-based scanning detects every device on your LAN — hostname, vendor, MAC, OS fingerprint, latency, and bandwidth per device.

Port Scanning

Full SYN scan with service + version detection. Flags risky ports (RDP, Telnet, SMB, port 4444) and shows service banners.

CVE Scanner

Cross-references discovered services against known CVEs. Shows severity, CVSS score, and remediation hints per device.

Traffic Analysis

Captures traffic patterns in real time. Shows top destinations, ASN + org info, byte counts, and GeoIP mapping of all external connections.

Anomaly Detection

Detects SSH brute-force, ARP spoofing, port scans, DNS tunneling, watchlist country connections, and traffic spikes in real time.

Real-Time Alerts

WebSocket-powered live alert feed with severity classification (critical/high/medium/low). CVE alerts, threat intel, and brute-force detection.

Quick Start

Up and running in 2 minutes

One Docker Compose command and your entire stack is live.

1

Clone

git clone https://github.com/SaharHalili95/netsentinel.git cd netsentinel
2

Configure

Set your network range in .env

cp .env.example .env # Edit SCAN_NETWORK=192.168.x.0/24
3

Launch

docker-compose up
4

Open dashboard

Navigate to http://localhost:3000 — scanning starts automatically.
